Le message Could not find the table 'public.compta_comptes' (ou « schema cache ») indique que le script SQL de comptabilité n’a pas encore été exécuté sur ce projet Supabase.

  1. Ouvrez le tableau de bord Supabase → votre projet → SQL Editor
  2. New query → copiez-collez tout le fichier compta_generale_schema.sql à la racine du dépôt (ou depuis votre clone)
  3. Cliquez Run (succès attendu), puis revenez ici et cliquez Charger.

En Suisse, le plan comptable de référence pour les PME est le Kontenrahmen KMU (« plan comptable suisse PME »). Il est publié gratuitement en PDF (français, allemand, italien) par la Confédération :

PDF — Plan comptable suisse PME (français, SECO / kmu.admin.ch)

Le fichier swiss-kmu-kontenplan.json dans ce dossier est un extrait structuré pour démarrer dans Colixo (pas le plan exhaustif). Les numéros doivent correspondre à votre paramétrage Bexio / fiduciaire.

Exécutez compta_generale_schema.sql dans Supabase pour créer les tables. Vous pouvez compléter avec l’import KMU ci-dessus ou des comptes manuels.
